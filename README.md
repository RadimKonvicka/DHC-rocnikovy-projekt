# DHC Records Vault - Ročníkový projekt
Webová aplikace v Djangu pro správu hudebního archivu (DHC scéna).

## Funkcionalita
- Evidence interpretů, alb a konkrétních tracků.
- Administrační rozhraní pro správu dat.
- Dynamické stránky: Index, Seznam alb, Detail alba.

## Spuštění
1. Aktivovat venv: `venv\Scripts\activate`
2. Instalovat závislosti: `pip install -r requirements.txt`
3. Spustit server: `python manage.py runserver`
